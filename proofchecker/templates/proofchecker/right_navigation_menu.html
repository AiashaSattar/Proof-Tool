How to use:
Atomic sentences:
Atomic sentences can be represented with uppercase letters A-Z
Premises:
Separate premises using semicolons (e.g. "A; B").
Entering symbols:
To enter math symbols, use the bold escape commands below while typing.

Disjunction \or ∨
Negation \not ¬
Implication \implies →
Biconditional \iff ↔
Contradiction \falsum ⊥
Universal Quantifier \forall ∀
Existential Quantifier \exists ∃
Domain (Set) Membership \in ∈
Row numbering:
